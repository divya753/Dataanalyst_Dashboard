# Sales Insights Data Analysis using Power BI

## Project Overview
This project analyzes sales data using Power BI to understand revenue and business performance.

## Dataset Source
Sales data stored in a MySQL database. Data was extracted using SQL queries and exported as CSV files.

## Dataset Details
- Customers  
- Products  
- Markets  
- Transactions  
- Date  

## Data Cleaning
- Removed duplicates  
- Handled missing values  

## Dashboard Features
- Revenue KPIs  
- Sales trends  
- Top customers and products

## Tools Used
- Microsoft Power BI  
- MySQL  
- SQL  


## How to Use
1. Download the .pbix file  
2. Open it using Power BI Desktop  
3. Use filters and slicers to explore the dashboard  

## Key Insights
- Identified top-performing products contributing highest revenue  
- Observed sales trends over different time periods  
- Found high-revenue markets and customers
